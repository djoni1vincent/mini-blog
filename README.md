# Mini Blog

A learning project based on **The Flask Mega-Tutorial**. This project is designed to build a functional blog application while mastering core web development concepts.

## Learning Outcomes

Upon completion, you will understand:

- Flask fundamentals and application structure
- Database design with SQLAlchemy ORM
- User authentication and authorization
- Form handling and validation
- Templating with Jinja2
- Database migrations
- RESTful API principles
- Testing and debugging
- Deployment best practices
